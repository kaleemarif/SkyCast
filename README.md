# 🌦 SkyCast

SkyCast is a modern animated weather application that provides real-time weather updates with a premium glassmorphism UI, smooth animations, and dynamic weather effects.

Built using HTML, CSS, JavaScript, and OpenWeather API.

---

## ✨ Features

- 🔍 Search weather by city
- 📍 Current location weather support
- 🌡 Real-time temperature updates
- 💨 Wind speed, humidity & visibility
- 🌤 Dynamic weather icons
- 🌧 Animated weather effects
- ✨ Glassmorphism UI
- 📱 Fully responsive design
- ⚡ Smooth animations and transitions
- 🚨 Error handling & loading states

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
- Font Awesome
