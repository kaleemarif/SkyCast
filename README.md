🌦️ SkyCast — Advanced Weather Web Application

SkyCast is a modern responsive weather web application built using HTML, CSS, and JavaScript. It provides real-time weather information with a premium glassmorphism UI, dynamic weather effects, interactive forecasts, AQI monitoring, maps, and voice search functionality.

🚀 Live Demo

https://kaleemarif.github.io/SkyCast/

✨ Features

- 🌤️ Real-Time Weather Data
- 📅 5-Day Weather Forecast
- ⏰ Hourly Forecast
- 🌫️ Air Quality Index (AQI)
- ⭐ Favorite Cities
- 🗺️ Interactive Weather Maps
- 🎤 Voice Search
- 🌧️ Dynamic Weather Animations
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- ⚡ Fast & Lightweight

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
- Font Awesome
